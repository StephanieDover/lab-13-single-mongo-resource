#how to install  

 * to install in terminal:  npm run start-db  

 * look at package.json for dependancies.   

#starting the server

 * to run server in terminal: npm run start  

#routes

 * POST: '/api/products'  
 * PUT: '/api/products/:id'  
 * GET: '/api/products/:id'  
 * DELETE: '/api/products/:id'
